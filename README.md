# Darwin Phishing URL Analyzer

## Overview

This project analyzes URLs for common phishing indicators used by cybercriminals.

The tool checks for:

- Insecure HTTP connections
- Suspicious phishing keywords
- Numeric/IP-based domains
- Risk scoring
- High Risk / Medium Risk / Low Risk classification

---

## Features

- Detects suspicious URLs
- Identifies insecure HTTP websites
- Flags phishing keywords
- Simple risk scoring
- Beginner SOC Analyst project

---

## Technologies

- Python 3

---
